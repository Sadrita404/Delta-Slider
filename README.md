### About

Delta Slider is an automated track system that moves a camera smoothly using a built-in motor. It eliminates the shakiness of manual sliding and allows videographers to program precise, repeatable camera movements for video interviews, product B-roll, and complex time-lapse shots

### Why I chose this project?
Behind choosing this project is because I made project and when it's time to get some good pictures and I want to get in time lapse of that it is hard to get it from the hand totally manually, so I have designed this motorised camera slider which can easily automate the process which I done using hand and using this, it would be much more efficient for me to get those videos

### How to use this project

This project can be used in multiple ways, but the best ways to use it for taking long or short time lapse of something. Since it's an  motorised one, it would give smooth slider effect on the video.
